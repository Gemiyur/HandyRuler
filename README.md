# Экранная линейка
Экранная линейка с измерением углов и лупой.

За основу взят репозиторий [doug24/Ruler](https://github.com/doug24/Ruler).  
Форк репозитория: [Gemiyur/Ruler](https://github.com/Gemiyur/Ruler).

## Изменения в оригинале
- Русский язык интерфейса
- Изменён макет окна настроек
- Размер шрифта окна настроек по умолчанию 12 пунктов
- В меню и настройках сантиметры перед дюймами

## Возможности линейки
- Единицы измерения:
  - Пиксели
  - DIP (пиксели, не зависящий от устройства)
  - Пункты
  - Сантиметры
  - Дюймы
  - Проценты размера экрана
- Горизонтальная или вертикальная ориентация линейки
- Положение нулевой точки слева (сверху) или справа (снизу) линейки
- Закрепление линейки по краям экрана
- Точная настройка положения и длины линейки и положения курсора мыши с клавиатуры
- Установка маркеров
- Измерение углов
- Встроенная лупа с регулируемым масштабом
- Отслеживание положения курсора в любом месте экрана
- Выбор цветовой темы

Пользовательская конфигурация сохраняется с помощью [Config.Net](https://github.com/aloneguid/config).
